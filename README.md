# Vite Antd RJSF example

Steps to create:

```sh
brew install pnpm
nvm install

pnpm create vite . --template react-ts

pnpm install antd --save
pnpm install @rjsf/core @rjsf/utils @rjsf/antd @rjsf/validator-ajv8 --save

# Updated src/App.tsx to render RJSForm
```

Run

```sh
pnpm run dev
```
